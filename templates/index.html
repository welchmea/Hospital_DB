Hospital Database Directory

Departments
This page allows you to view hospital Departments, add a new Department, update a Department, and remove a hospital Department.
Employees
This page allows you to create a new Employee, view current Employees, delete an existing Employee, and Update any existing Employee.
Jobs
This page allows you view update, delete, and add new Employee Jobs.
Schedules
This page allows you to view, delete and add new Employee Schedules.
Employment Types
This page allows you to view and delete the various Employment Types offered to Employees and add a new status if required.
Department Jobs
This page is an intersection table betweeb Departments and Jobs. It allows viewing of the different Jobs currently working in all of the Departments and the option to add a new combination, delete from, as well as update.
Department Employment Types
This page is an intersection table between Departments and EmploymentTypes. It allows for viewing current data, deletion, and adding a new EmploymentType to Departments.
